# Airline MVP

**«Airline MVP»** — простой экран для MVP сервиса дополнительных услуг для авиапассажиров. Интерфейс для выбора услуг, добавления их в корзину, расчёта итоговой стоимости и оформления заказа.

## Стек

React 18+, TypeScript, CSS3 (БЭМ)

## Быстрый старт
```
npm install
npm run dev
```

## Что реализовано

- React + TypeScript с функциональными компонентами  
- Управление состоянием через useState  
- Список услуг с ценами и описаниями  
- Добавление услуг в корзину  
- Динамический расчёт итоговой суммы  
- Мобильный-first дизайн
- БЭМ
